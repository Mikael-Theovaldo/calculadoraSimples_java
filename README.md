# calculadoraSimples_java
Calculadora simples com a linguagem de programação Java.


# Sobre o Projeto

Uma calculadora simples que realiza as quatro operações matemáticas básicas (soma, subtração, multiplicação e divisão).

Este projeto foi desenvolvido como parte dos meus estudos iniciais em programação Java, demonstrando conceitos fundamentais como:
- Estruturas de controle (loops e condicionais)
- Métodos e funções
- Tratamento de exceções
- Entrada e saída de dados

# Funcionalidades

- Soma de dois números
- Subtração de dois números
- Multiplicação de dois números
- Divisão de dois números (com proteção contra divisão por zero)
- Operações contínuas até o usuário decidir sair
- Validação de entradas

# Versões Implementadas

### Versão 1: Calculadora Simples (com métodos)
Implementação utilizando métodos separados para cada operação, demonstrando:
- Uso de métodos estáticos
- Retorno de valores
- Tratamento de exceções com `throw`
- Comparação entre método com retorno e método void

### Versão 2: Calculadora Básica
Implementação mais direta, com toda lógica no método `main`, ideal para entender o fluxo básico do programa.

# Tecnologias Utilizadas

- **Java** - Linguagem de programação

# Como Executar

### Pré-requisitos
- Java JDK 8 ou superior instalado

### Passos para execução

1. Clone o repositório:
```bash
git clone https://github.com/Mikael-Theovaldo/calculadoraSimples_java.git
```

2. Navegue até o diretório do projeto:
```bash
cd calculadoraSimples
```
ou 

```bash
cd calculadoraBasica
```

3. Compile o código:
```bash
javac src/exercicios/CalculadoraSimples.java
```
ou

```bash
javac src/exercicios/CalculadoraBasica.java
```

4. Execute o programa:
```bash
java src.exercicios.CalculadoraSimples
```
ou 

```bash
java src.exercicios.CalculadoraBasica
```

# Como Usar

1. Ao iniciar o programa, escolha uma das opções:
   - `1` para Soma
   - `2` para Subtração
   - `3` para Multiplicação
   - `4` para Divisão
   - `0` para Sair

2. Digite o primeiro número quando solicitado

3. Digite o segundo número quando solicitado

4. O resultado será exibido automaticamente

5. Você pode continuar fazendo operações até escolher a opção `0` para sair
